Digital mirror

Project Description 3 stages :Symmetrical Reflection—--Mechanical Reflection—--Fade / Dissolution Stage 1: SoftMirror • Particles gently move when the mouse approaches • Mouse interaction creates a soft, flowing effect • Represents the “soft and light” reflection stage Stage 2: MechanicalMirror • Pixels are transformed into geometric shapes (circle, square, triangle) based on brightness • Cold color (blue/cyan tones) emphasizes a mechanical, algorithmic aesthetic • Static layout, no mouse interaction • Represents the “structured, algorithmic reflection” stage Stage 3: FadeMirror • Inherits the geometric classification of Stage 2 • Gradually fades into gray/black as time progresses • The algorithm's response to movement slowly disappears (“identity dissolves”) • Represents the “reflection losing interest” stage, emphasizing impermanence and fade-out
